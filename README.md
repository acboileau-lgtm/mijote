# Mijoté

Prototype d’application pour organiser les repas de la semaine.

## Ouvrir l’application

Double-cliquez sur `index.html`. Aucune installation ni création de compte n’est nécessaire.

Les recettes, menus, courses et aliments ajoutés sont conservés dans le stockage local du navigateur.

## Fonctions incluses

- planning des déjeuners et dîners ;
- ajout, recherche et filtrage des recettes ;
- planification manuelle ou automatique ;
- liste de courses interactive et classée par rayon ;
- inventaire du frigo avec dates limites ;
- interface adaptée à l’ordinateur et au téléphone.
